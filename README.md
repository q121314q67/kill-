# 圣奥里杀戮系统 v9.7

一个功能强大的 Roblox 圣奥里脚本库，提供近战/远程自动化、ESP透视、秒互动等实用工具。

## 🚀 快速开始

### 方法 1: 直接加载 (推荐)

复制以下代码到 Roblox 脚本执行器：

```lua
loadstring(game:HttpGet("https://github.com/q121314q67/kill-/raw/main/main.lua", true))()
```

### 方法 2: 加载完整系统

```lua
loadstring(game:HttpGet("https://github.com/q121314q67/kill-/raw/main/SaintAuri_KillSystem.lua", true))()
```

### 1. 近战杀戮系统

- ✅ 好友检测

- ✅ **玩家透视 (ESP强化版)**

## 🎮 UI 操作

### 侧边栏
- 右边屏幕边缘有一个 **侧边栏触发器**
- 点击展开侧边栏菜单
- 5秒自动隐藏（非活动时）

### 快捷键绑定
1. 在各功能面板中长按 (0.8秒) 开关按钮
2. 在屏幕上点击放置快捷键
3. 支持多个快捷键同时存在
4. 长按快捷键 (0.8秒) 可删除
5. 短按快捷键快速开关功能

### 快捷键图标
- ⚔️ 近战杀戮
- 🔫 远程杀戮  
- 👁️ 实用工具
- ⚙️ 设置

## 🔧 配置选项

### 检测模式
| 选项 | 说明 |
|------|------|
| 好友检测 | 启用则自动跳过好友 |
| 可见性检测 | 启用则检查目标是否被遮挡 |
| 队伍选择 | 仅攻击指定队伍的玩家 |

### 攻击模式
| 选项 | 说明 |
|------|------|
| 多部位打击 | 近战时攻击多个身体部位 |
| 仅爆头 | 远程时只瞄准头部 |
| 散弹多发 | 远程时发送多发子弹 |

每个玩家上方会显示：
```
玩家名字 [当前血量/最大血量] [锁定状态]
例: Player123 [50/100] [近战锁]
```

### 颜色标识
- 🟢 默认队伍颜色
- 🔴 近战锁定中
- 🔵 远程锁定中

## ⚙️ 高级设置

### 配置修改
编辑 `SaintAuri_KillSystem.lua` 中的 `Config` 表：

```lua
local Config = {
    Melee_Enabled = false,           -- 近战启用
    Melee_MultiHit = true,           -- 多部位打击
    Ranged_Enabled = false,          -- 远程启用
    Ranged_AutoHeadshot = true,      -- 自动爆头
    ESP_Enabled = false,             -- ESP启用
    InstantInteract_Enabled = false, -- 秒互动启用
}
```

## 🐛 故障排查

### 脚本加载失败
1. 确保网络连接正常
2. 检查 GitHub 是否可访问
3. 尝试清除浏览器缓存

### 功能不工作
1. 检查是否正确启用了相应功能
2. 查看游戏控制台错误信息 (F9)
3. 确保目标在有效范围内
4. 验证 Remote 事件是否存在

### 性能问题
1. 减少同时启用的功能数量
2. 关闭 ESP 和其他渲染功能
3. 重启游戏和脚本
